TROUBLESHOOTING
================================================================================

COMMON ISSUES:

1. "Element not found"
   → Update XPath locators in PageLocators.java
   → See LOCATOR_UPDATE_GUIDE.md

2. "Login fails"
   → Verify credentials in BaseTest.java
   → Check application is running
   → See TROUBLESHOOTING_REFERENCE.md

3. "Timeout errors"
   → Increase wait time in BaseTest.java
   → Check element visibility
   → See TEST_EXECUTION_GUIDE.md

4. "Tests fail together but pass individually"
   → Ensure tests are independent
   → Check tearDown() closes browser
   → See TROUBLESHOOTING_REFERENCE.md

For more issues, see TROUBLESHOOTING_REFERENCE.md
